MRErrorBuilder.m

Overview

Concrete builder of NSError instances.

Adds accessors for AFNetworkingOperationFailingURLRequestErrorKey and AFNetworkingOperationFailingURLResponseErrorKey user info values.

Warning: This extension requires the AFNetworking library. Add an AFNetworking.h import to the header prefix of the project.

Adds accessors for AVError userInfo values.

Warning: This extension requires the AVFoundation framework. Add a AVFoundation/AVFoundation.h import to the header prefix of the project.

Adds accessors for FBError userInfo values.

Warning: This extension requires the Facebook SDK framework. Add a FacebookSDK.h import to the header prefix of the project.

Adds accessors for AVError userInfo values.

Warning: This extension requires the Core Data framework. Add a CoreData/CoreData.h import to the header prefix of the project.

Adds accessors for @"JKAtIndexKey" and @"JKLineNumberKey" user info values.

Warning: This extension requires the JSONKit library. Add an JSONKit.h import to the header prefix of the project.

Adds a convenience constructor that takes an exception as parameter.

Creates and initializes an MRErrorBuilder object for a given domain and code.

+ (instancetype)builderWithDomain:(NSString *)domain code:(NSInteger)code

Parameters

domain

The error domain.

code

The error code for the error.

Return Value

Builder object initialized for domain with the specified error code and some predefined localized description.

Discussion

Creates and initializes an MRErrorBuilder object for a given domain and code.

The builder is given a predefined localized description, according its domain and code; but this string may not be adequate for being presented to the user.
